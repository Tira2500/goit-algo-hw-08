# goit-algo-hw-08

recursive_min_search.py: Реалізація алгоритму рекурсивного пошуку найменшого значення у двійковому дереві пошуку (BST) або AVL-дереві. Алгоритм використовує властивість дерева, де кожен лівий нащадок менший за батьківський вузол, забезпечуючи швидкий пошук мінімального елемента.

tree_sum.py: Функція для обчислення суми всіх значень у двійковому дереві пошуку або AVL-дереві. Алгоритм здійснює рекурсивний обхід усіх вузлів структури, додаючи значення кожного вузла до загальної суми, що дозволяє отримати результати для будь-якого непорожнього дерева.

min_cost_cables.py: Рішення задачі на мінімізацію витрат при об'єднанні мережевих кабелів. Алгоритм використовує структуру даних «мінімальна купа» (min-heap) з бібліотеки heapq для того, щоб на кожному кроці ефективно вибирати та з'єднувати два найкоротші кабелі, мінімізуючи загальну суму витрат на з'єднання.
